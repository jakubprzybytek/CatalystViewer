Current stats show graphs that display constant part of interest rate. Each graph displays data for specific interest rate const part, for example: WIBOR 3M, EURIBOR 6M OR Const. Additionally, values for constant part of interest rate are divided into four colour coded buckets: 25th percentile, 50th percentile, 75th percentile and 100th percentile.

New stats show graphs with bonds data grouped by interest base type, one chart per interest base type (same grouping as today). Each chart is a 2D scatter plot where:
- X axis — bond's issue date (`firstDayTs`), plotted as exact dates; axis tick labels are sparse to avoid clutter.
- Y axis — the constant part of the interest rate (`interestConst`).
- Colour coding — four buckets (25th / 50th / 75th / 100th percentile) computed per interest base type group, using percentile thresholds derived from `allBondReports` (global baseline, not affected by current filter).

The charts display only `filteredBondReports`. If no issuers are selected, all bonds are included.

The button label 'Show New Stats' and the feature name 'new stats' are temporary, for the duration of development.

Implementation uses Recharts (already a project dependency) as the charting library.

Acceptance Criteria
1. Next to 'Show Stats' there is a toggle button 'Show New Stats' that expands a panel with the new stats charts.
2. New stats consider bonds that are currently filtered. If no issuers are selected then bonds from all issuers are taken into account.
3. There is one scatter chart per interest base type present in the filtered bond set.
4. X axis shows bond issue date; tick labels are sparse enough not to clutter the axis.
5. Y axis shows the constant part of the interest rate.
6. Each point is colour coded according to the quartile bucket it falls into, where the quartile thresholds are computed from all bonds (not just filtered), per interest base type group.
